Syntax :< a href = "URL" >URL Name < /a >

The attribute target is used to tell the browser where to open the Link on which the user hit click.

    • _blank : - _blank tell the browser to open the link in new tab.
      _self : - _self tell the browser to open the link in default or same tab.
      _parent : - _parent tell the browser to open the link in it's parent tab.
      _top : - _top the topmost browsing context ,It is on the top of all the parents.
